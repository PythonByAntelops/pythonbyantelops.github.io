CURSO "Python para Administradores de Sistemas y DevOps"

Curso de 5 días orientado a Administradores de Sistemas y DevOps, con o sin experiencia en Python. El curso incluye numerosos ejercicios prácticos orientados a la administración de sistemas y la automatización de tareas.

Calendario

Contenido

Los objetivos formativos del curso incluyen los siguiente contenidos:

  • Fundamentos del lenguaje
  • Herramientas de desarrollo
  • Ciclo de desarrollo
  • Pruebas en Python
  • Python para automatización
  • Patrones y expresiones regulares para búsquedas
  • Trabajando con ficheros CSV y JSON
  • Planificando tareas y ejecutando programas
  • Python para reporting
  • Concurrencia y paralelismo
  • Python para AWS

Prerrequisitos

Experiencia previa básica con algún lenguaje de programación o con scripting.
